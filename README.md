# MPCSim
This MATLAB-based project provides a framework for **Model Predictive Control (MPC) simulation**, enabling the construction and solution of MPC optimization problems. It can be used as a general-purpose controller within closed-loop systems.

# Dependencies
- MATLAB R2015+
- OSQP
Installation: <https://osqp.org/docs/get_started/matlab.html>
Usage: <https://osqp.org/docs/interfaces/matlab.html>
